Usage

In a terminal (e.g. in Linux) do the following:

--------------------------------------------------------------------

To run the Python program use
python3.4 ford_fulkerson.py < input.txt

--------------------------------------------------------------------

To run the C++ program you have to first compile it with
g++ ford_fulkerson.cpp -std=c++11 -O2 -o ford_fulkerson.out -Wall

and then run the executable with 
./ford_fulkerson.out < input.txt

-------------------------------------------------------------------

where input.txt contains the input data and is read via "i/o redirection", see
https://en.wikipedia.org/wiki/Redirection_(computing)

In the folder there are six different input files.
